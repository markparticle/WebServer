### 找不到mysql/mysql.h

没有mysql开发包，安装mysql开发包

```c++
../code/server/../pool/sqlconnpool.h:9:10: fatal error: mysql/mysql.h: No such file or directory
    9 | #include <mysql/mysql.h>
```
**解决方案：**
```bash
sudo apt-get install libmysqlclient-dev
```
